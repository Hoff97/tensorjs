# Development

## Setup

Make sure you have [wasm-pack](https://github.com/rustwasm/wasm-pack) installed, then

```sh
$ npm run build:wasm
$ npm install
```

## Build

```sh
$ npm run build
```

## Testing

```sh
$ npm run test
```

There are tests for the rust and js implementation:
```sh
$ npm run test:wasm
$ npm run test:js
```
